# Intro
Опытный менеджер замечает проблему раньше, чем она становится катастрофой. Не по жалобам — по слабым сигналам. Один человек в команде находится на грани выгорания. Найди его — пока не стало поздно.

# Instruction
На экране — восемь карточек с профилями членов команды. Каждую карточку можно развернуть тапом и изучить детали: количество задач, встречи в неделю, время последнего отпуска, короткие цитаты из рабочего чата, рабочие часы.

Изучи все профили и найди одного человека с признаками выгорания. Обрати внимание на тихие сигналы — не на громкие слова.

# Moral
Выгорание не кричит — оно молчит. Менеджер, который замечает слабые сигналы, сохраняет и людей, и проекты.
