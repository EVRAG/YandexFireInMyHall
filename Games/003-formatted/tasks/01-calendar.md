# Intro
Утро понедельника. На столе — куча задач, три дня впереди и уже несколько встреч в календаре. Проджект-менеджер не делает всё подряд — он расставляет приоритеты.

Изучи календарь и расставь все задачи по дням так, чтобы ничего не сорвалось. Предстоит изучить страницу-календарь с пустыми слотами и несколькими встречами, расставить встречи в пустые слоты с учётом особенности каждой встречи.

# Instruction
Экран разделён на четыре зоны:
- Слева — узкая колонка с пулом из 7 карточек-задач для распределения
- Справа — три колонки-календаря: Пн 14 марта / Вт 15 марта / Ср 16 марта, каждая с временными слотами с 9:00 до 18:00
- В календаре уже размещены три зафиксированные карточки-якоря (с замочком), которые не перемещаются

Перетаскивай карточки из пула в подходящие временные слоты. По тапу на любую карточку — всплывает подсказка с описанием ограничений.

# Moral
Планирование — это не просто расстановка задач по времени. Это учёт бесконечного количества факторов. Хороший проджект-менеджер видит не только свой календарь — но и календари всей команды.
