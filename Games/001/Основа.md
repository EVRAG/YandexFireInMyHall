**Тач-панель “Креативный трек”**

На экране во время бездействия тач-панели не отображается специальная заставка. Вместо него –  главное меню, в котором представлены разделы:

* \[текстовое описание\]  
* \[задания\]  
* \[видео\]  
* \[тест\]

  Примечание: 


  – если эти пункты не помещаются в статичный экран, можно предусмотреть возможность “пролистывания” страницы вниз, но важно очень понятно отобразить, что такое пролистывание возможно (чтобы пользователь сразу понимал, что есть еще контент)

  – изначально не озаглавливаем подразделы текстовыми заголовками, вместо этого можно сделать структуру интуитивно понятной через дизайн (цвет, визуальные элементы) (обсуждаемо).

| Главное меню |  |  |
| :---- | :---- | :---- |
| Блок описания  |  |  |
| Текстовый блок-описание креативного трека как профессионального направления (в целом). Оно отображается по умолчанию и включает возможность перехода \[по тапу\] на уточненное описание конкретных профессий: \[Графический дизайнер\] \[UX-дизайнер\]  По тапу на любую из профессий  мы попадаем на страницу с описанием. Из этого описания можно \[➜\] на следующее описание внутри уровня, а также нажать кнопку \[↩назад\]. Примечание: все эмодзи и символы приведены условно, дизайнить можно другие \- главное, чтобы был интуитивно понятный интерфейс. |  |  |
| Блок заданий  |  |  |
| Блок включает крупные кнопки с названиями заданий, \[тап\] по которым открывает конкретное задание. Важно отобразить на каждой кнопке  1\) относится ли задание к групповым или индивидуальным – например, используя интуитивно понятные иконки вроде 👤и 👥 2\) время прохождения задания 🕓в минутах По тапу на любое из заданий  мы попадаем на страницу с заданием.  В любой момент можно нажать кнопку \[↩назад\], а по завершению конкретного задания по кнопке \[➜\] на следующее задание внутри блока |  |  |
