**Тач-панель “Менеджмент”**

На экране во время бездействия тач-панели не отображается специальная заставка. Вместо него –  главное меню, в котором представлены разделы:

* \[текстовое описание\]  
* \[задания\]  
* \[видео\]  
* \[тест\]

  Примечание: 


  – если эти пункты не помещаются в статичный экран, можно предусмотреть возможность “пролистывания” страницы вниз, но важно очень понятно отобразить, что такое пролистывание возможно (чтобы пользователь сразу понимал, что есть еще контент)

  – изначально не озаглавливаем подразделы текстовыми заголовками, вместо этого можно сделать структуру интуитивно понятной через дизайн (цвет, визуальные элементы) (обсуждаемо).

| Блок описания  |  |  |
| Текстовый блок-описание менеджмента как профессионального направления (в целом). Оно отображается по умолчанию и включает возможность перехода \[по тапу\] на уточненное описание конкретных профессий:  Технический менеджер Продакт-менеджер Проджект-менеджер  По тапу на любую из профессий  мы попадаем на страницу с описанием. Из этого описания можно \[➜\] на следующее описание внутри уровня, а также нажать кнопку \[↩назад\]. Примечание: все эмодзи и символы приведены условно, дизайнить можно другие \- главное, чтобы был интуитивно понятный интерфейс. |  |  |
| Блок заданий  |  |  |
| Блок включает крупные кнопки с названиями заданий, \[тап\] по которым открывает конкретное задание.  “[День не резиновый ( я календарь переверну)”](?tab=t.hle9ms2o2aa9)  \[👥; 🕓5\] [“Признаки выгорания”](?tab=t.c9a64zty661z) \[👥; 🕓5\] [“Защита бэклога”](?tab=t.ok2jwxih40ep)\[👤, 🕓3\] [“Собери запуск”](?tab=t.fi8uxqpd34zb) \[👤, 🕓3\] [“Прочитай между строк”](?tab=t.1kk9ny4ats6n)\[👤, 🕓4\]  Примечание: у каждого задания есть параметры. Важно отобразить на каждой кнопке  1\) относится ли задание к групповым или индивидуальным – например, используя интуитивно понятные иконки вроде 👤и 👥 2\) время прохождения задания 🕓в минутах По тапу на любое из заданий  мы попадаем на страницу с заданием.  В любой момент можно нажать кнопку \[↩назад\], а по завершению конкретного задания по кнопке \[➜\] на следующее задание внутри блока |  |  |
