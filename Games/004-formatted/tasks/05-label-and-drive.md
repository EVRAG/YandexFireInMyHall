# Intro
Беспилотное такси видит мир глазами камер. Но прежде чем оно научилось различать пешехода и столб — тысячи людей вручную объясняли ему: вот машина, вот человек, вот светофор. Без этой работы автопилот просто не смог бы тронуться с места. Попробуй сам.

Увидь мир глазами видеорегистратора и помоги роботакси тронуться с места, присвоив объектам вокруг правильные теги.

# Instruction
На экране — стоп-кадр с видеорегистратора роботакси. Объекты, которые нужно разметить, подсвечены мягким пульсирующим контуром.

Для каждого объекта:
1. Тапни на него
2. Обведи рамкой (drag по экрану)
3. Выбери подходящий тег из выпадающего списка

Счётчик прогресса: «Размечено: 0/5»

Все объекты размечены верно — такси трогается с места! Есть ошибки — такси стоит, неверные объекты подсвечиваются красным.
