# Intro
Вам нужно создать алгоритм, который предскажет, станет ли новый фильм хитом. [Нейросеть]{tooltip: "Нейросеть — программа, которая учится на примерах, как мозг учится на опыте"} нужно натренировать, но она не может «скушать» всё подряд. Нужно выбрать правильные факторы.

# Instruction
На экране появляются факторы один за другим. Для каждого нужно решить: влияет ли он на кассовый успех фильма?

- «Влияет» — фактор будет использован для обучения модели
- «Не влияет» — фактор отбрасывается

Вверху экрана — индикатор точности модели (начальное значение 50%). Правильные решения повышают точность, ошибки — снижают.
